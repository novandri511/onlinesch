PERANGKAT LUNAK PENGOLAH KATA

Aplikasi pengolah kata (word processor) merupakan software / perangkat lunak yang digunakan untuk mengolah kata dan berkaitan dengan dokumen, seperti tahapan membuat, mengedit, memodifikasi, menyunting maupun mencetak dokumen, banyak digunakan untuk membuat naskah laporan, artikel, brosur, karya tulis, surat menyurat, dan lain-lain. Berikut beberapa contoh Perangkat Lunak Pengolah Kata :


1.Microsoft Word
Microsoft word merupakan program pengolah kata yang paling banyak digunakan oleh semua orang, Microsoft Word mempunyai kemampuan yang sangat bagus dalam membuat berbagai jenis dokumen mulai dari buku, buku memo, surat, berbagai jenis dokumen-dokumen dan lainnya.Selain itu Microsoft Word dapat digunakan dengan cara mudah dengan kata lain “user friendly” Karena Microsoft Word dilengkapi dengan tampilan berupa grafis yang dilengkapi dengan tombol-tombol dan menu menu yang membuat kita bekerja dengan sangat mudah.


2. Lotus Word Pro
Lotus word pro merupakan aplikasi untuk pengolahan kata yang dikeluarkan oleh IBM, apikasi ini dikeluarkan berbarengan dengan aplikasi office lainnya yang dikenla dengan naman Lotus smartsuite. Kelebihan yang dimiliki oleh Lotus Word Pro adalah kemampuannya untuk mempertukarkan file file dokumen dengan Microsoft Word, Lotus word pro juga dapat membuka file file microsoft word aplikasi pengolahan kata lainnya. Word Pro dilengkapi dengan fasilitas Helpfull Menu Finder yang membantu pengguna untuk menemukan seluruh perintah yang sesuai dengan perintah wordperfect.


3. Corel WordPerfect
Aplikasi pengolahan kata lainnya yaitu wordperfect, yang merupakan aplikasi program yang dapat dijalankan dikomputer dengan sistem operasi windows. Wordperfect versi terakhir yaitu perfect office X3 yang dapat dijalankan pada sistem operasi windows xp, windows 2008, windows 98. Namun terdpat kekurangan pada wordperfect yaitu perfect office 3 membutuhkan spesifikasi hardware yang cukup tinggi yaitu prosesor Pentium 3 atau sederajat dengan memori 128 MB. Wordperfect office X3 memiliki kemampuan untuk membaca file file yang dikerjakan pada jenis aplikasi pengolahan kata lain, seperti pada aplikasi microsoft word, microsoft excel dan microsoft powerpoint.


4. Star Office Writter
Star office writer memiliki kesaman dengan microsoft word, yang merupakan aplikasi perangkat lunak yang bersifat multiplatform, yaitu aplikasi dapat dijalankan diberbagai macam sistem operasi baik linux, windows, unix, solaris dan sistem operasi lainnya. Perbedaan yang mendasar antara Starr Office writter dengan Microsoft office yaitu terletak pada file ekstensinya. Jika pada Ms.word menggunakan file ekstensi *doc, pada Star Office Writter meenggunakan file ektensi *odt. Namun jenis file ektensi *odt masih dapat dirubah menjadi file ektensi *doc.
